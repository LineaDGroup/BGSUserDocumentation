# BioBarica Global System

**Gracias por utilizar BioBarica Global System.**

Existen tres tipos principales de usuarios del sistema. Seleccione uno para ver la documentación relacionada:

> [Distribuidor](/es/distributor/)

Un usuario y empresa que vende Cámaras Hiperabáricas de BioBarica en diferentes lugares del mundo.

> [Propietario](/es/owner/)

Un usuario que compra una cámara y la utiliza en un centro médico.

> [Profesional](/es/professional/)

Un usuario que trabaja en una instalación médica de un propietario.
